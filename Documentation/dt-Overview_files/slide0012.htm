Data Test Program (dt)

Generic DT Options
nLimit I/O by data limit (limit= option).
nLimit I/O by record limit (count= or records= options).
nLimit I/O by volume limit (volumes= and vrecords= options).
nLimit I/O by runtime limit (runtime=timeString option).
nControl of error limit (errors= option, default is 1).
nControl of pass limit (passes= option, default is 1).
nControl of multiple processes (procs= option).
nControl of process behavior (oncerr=abort or continue).
nControl of device types, size, etc. (dtype= and dsize= options).
nPOSIX Asynchronous I/O (enable=aio or aios=value options).
