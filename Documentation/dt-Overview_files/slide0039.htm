Data Test Program (dt)
3
Short History of DT
nOriginally developed while working at Compugraphic in the late 1980’s initially for testing tape drives on SUN/OS.
nWritten to replace Unix dd/cmp utilities.
nCommand syntax is similar to Unix dd.
nLater disks, serial/parallel lines, and other device type support was added.
nDeveloped to verify device drivers and for device qualification (new hardware).
nGoal was to be generic yet flexible enough to customize tests.
