Data Test Program (dt)
34
Serial Line Features
nSupports setting of flow control (flow=none,cts_rts, xon_xoff).
nSupports setting of parity (parity=even,odd, or none).
nSupports setting of speed (speed= option, none gives list).
nHandles line disciplines on Tru64 Unix.
nSingle line loopback tests (enable=loopback)
nDual line loopback tests (if=/dev/tty00 of=/dev/tty01)
nDual line between different hosts and different OS’s too!
nLimited Tru64 modem support (enable=modem, on Tru64).
nParallel loopback support too (if=/dev/pig or /dev/pog, Agfa).
