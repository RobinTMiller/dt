Error Triggers
nOption: trigger=type
nExecuted on failures or no-progress
nTrigger Types:
ubr = Execute a bus reset.
ubdr = Execute a bus device reset.
useek = Issue a seek to the failing lba.
ucmd:string = Execute string w/these args:
ustring dname op dsize offset position lba errno noprogt
u
uThe first three options require Scu in your PATH.
