Data Pattern Files
nThere are several predefined data pattern files included in the source kit, containing the following data patterns:
upattern_0 - psuedo-random pattern.
upattern_1 - psuedo-random pattern alternating with 0's.
upattern_2 - all 0s.
upattern_3 - all 1s (FFH).
upattern_4 - 16 bit shifting 1 in a field of 0s.
upattern_5 - 16 bit shifting 0 in a field of 1s.
upattern_6 - alternating 01 pattern (AAH).
upattern_7 - byte incrementing (OOH - FFH).
upattern_8 - encrypted data pattern.
upattern_9 - psuedo-random pattern1 alternating with 0's.
upattern_all – concatenation of all of the above patterns.
upattern_dedup - each pattern file written (bs=4k records=5).
nThese files are specified with the pattern file (pf=file) option.
